# How to use:
1. Obtain connection details from WS Application (Home > Admin > Integration)
2. In python code:
```python
from ws_sdk.web import WS
ws = WS(api_url="WS_URL", user_key="USER_KEY", token="ORG_TOKEN")
all_alerts = ws.get_alerts()
```
